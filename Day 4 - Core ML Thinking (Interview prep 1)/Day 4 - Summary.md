# 🧠 Key Concepts Covered

## 🔹 Model Selection Strategy

- Start with **Logistic Regression (baseline model)**
- Move to **Decision Tree** for non-linear relationships
- Use **Random Forest** for improved generalization

---

## 🔹 Understanding Model Behavior

- Linear vs Non-linear models  
- Overfitting vs Generalization  
- Bias-Variance Tradeoff  

---

## 🔹 Model Comparison Thinking

- Accuracy is not the only metric  
- Small improvements (e.g., 82% → 84%) must be validated  
- Trade-offs:
  - Accuracy vs Interpretability  
  - Simplicity vs Complexity  

---

## 🔹 Evaluation Metrics

### Classification:
- Accuracy  
- Precision  
- Recall  
- F1 Score  

### Regression:
- MAE (Mean Absolute Error)  
- RMSE (Root Mean Squared Error)  
- R² Score  

---

## 🔹 Cross Validation

- Used to ensure model performance is consistent  
- Reduces dependency on a single train-test split  
- Provides more reliable evaluation  

---

## 🔹 Imbalanced vs Balanced Dataset

- Balanced → Equal class distribution  
- Imbalanced → One class dominates  
- Accuracy can be misleading in imbalanced data  
- Use **F1 Score, Precision, Recall** instead
